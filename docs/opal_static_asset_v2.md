# **2. סוגי טקסט — זיהוי לפי מטרה, מבנה ומילות מפתח**

לפני ניתוח המבנה — יש לזהות את סוג הטקסט. לכל סוג: מטרה, סימני זיהוי, ומילות קישור אופייניות.

**סוג הטקסט**

**מטרה ומבנה**

**סימני זיהוי**

**מילות קישור אופייניות**

**שאלת זיהוי**

💬 טקסט טיעוני
(Argumentative)
מטרה: לשכנע. הכותב לוקח עמדה ומביא ראיות לתמיכה. תמיד יש טענה ראשית שניתן להסכים/לחלוק עליה.

נוכחות "אני טוען" / "יש לאסור" / "מחויב" / ביטויי עמדה מפורשים. יש ניסיון לשכנע.

לכן, אולם, על כן, בניגוד ל-, יש הטוענים... ברם, מכאן ש-

האם הכותב לוקח צד ומנסה לשכנע אותי?

📋 טקסט תיאורי
אובייקטיבי
מטרה: להעביר מידע עובדתי בלי לקחת עמדה. מתאר מצב, תופעה, ישות. שפה ניטרלית.

שפה ניטרלית: "X הוא..." / "המחקר מגדיר" / ללא פעלי דעה / ללא קריאה לפעולה. אפשר לאמת.

בנוסף, לדוגמה, כלומר, כגון, בתחומים שונים

האם הטקסט מתאר עובדות בלי לנקוט עמדה?

⚖ השוואה
וניגוד
(Comparison)
מטרה: להצביע על קווי דמיון ו/או הבדל בין שני עניינים. מבנה: A vs B לאורך פרמטרים.

מופיעים שני נושאים מרכזיים. מבנה של "מצד אחד... מצד שני" או פסקאות נפרדות לכל נושא.

לעומת זאת, ואילו, בניגוד ל-, גם... וגם, בדומה ל-, שניהם

האם הטקסט עוסק בשני דברים ומשווה ביניהם?

🔄 תיאור תהליך
(Process)
מטרה: להסביר כיצד משהו מתרחש — שלבים עוקבים. מבנה כרונולוגי או לוגי. שפה פעילה.

מספרי שלבים או מילות רצף: "ראשית", "לאחר מכן", "לבסוף". פעלים: "מתחיל", "עובר", "מסתיים".

ראשית, לאחר מכן, בשלב הבא, כתוצאה, לבסוף

האם הטקסט מסביר כיצד משהו עובד צעד אחר צעד?

🧩 זוויות שונות
על נושא
(Multi-perspective)
מטרה: להציג עמדות שונות על אותו נושא. לא לוקח צד — מניח ריבוי קולות. נפוץ בסקירת ספרות.

"יש הטוענים ש-" / "מנגד, חוקרים אחרים" / ציטוטים ממקורות שונים / אין מסקנה ברורה של הכותב.

מנגד, לחלופין, בד בבד, מנקודת מבט של, יש הגורסים

האם הטקסט מציג כמה עמדות שונות בלי לפסוק ביניהן?

🔧 בעיה
ופתרון
(Problem-Solution)
מטרה: להגדיר בעיה ולהציע פתרון. מבנה דו-שלבי: הצגת המצב הבעייתי → הצעת מענה.

פסקה ראשונה/שנייה: הגדרת הבעיה. פסקה שלישית ואילך: "על מנת לפתור", "הפתרון המוצע", "מומלץ ל-".

על מנת ש-, יש לשקול, הפתרון טמון ב-, בכדי ל-

האם הטקסט מגדיר בעיה ומציע מענה?

📰 כתבה
עיתונאית
(News/Feature)
מטרה: דיווח עובדתי + נרטיב. פתיחה בנתון/אנקדוטה, ואחר כך פיתוח. יש קול אנושי מאחורי הכתיבה.

פתיחה בסיפור / ציטוט / נתון מפתיע. שפה נגישה. מובאות ממקורות מזוהים. סגירה עם פנייה לקורא.

לדברי, על פי, "ציטוט", בין כה ובין כה, מסתבר ש-

האם הטקסט מספר סיפור ומדווח על מציאות?

_💡 שאלת הזיהוי המרכזית: "מה הכותב רוצה שאעשה לאחר הקריאה? להאמין לו? ללמוד תהליך? להשוות? להבין צדדים שונים?"_

# **3. מיקום הטענה — דדוקטיבי לעומת אינדוקטיבי**

לאחר זיהוי סוג הטקסט — היכן מופיעה הטענה הראשית?

**סגנון**

**הגדרה ומבנה**

**מילות מפתח מזהות**

**דוגמה / שאלת הבחנה**

דדוקטיבי
(טענה בפתיחה)
הטענה הראשית במשפט הראשון/שני. ההצדקות, ראיות, דוגמאות מגיעות אחריה כתמיכה.

"הכותב טוען" / "לדעת הכותב" / "מחקר זה יראה" — כבר במשפט הראשון
"הרשתות החברתיות פוגעות... מחקרים מראים ש-"
שאל: טענה במשפט 1?
אינדוקטיבי
(טענה בסגירה)
הפסקה פותחת בגורמים, ראיות, תיאור. הטענה מגיעה בסוף כמסקנה מתבקשת.

"לכן" / "אם כן" / "מכאן שניתן להסיק" / "מכל האמור" — בסוף הפסקה
"מחקרים מצביעים... נתונים מראים... לכן ניתן לטעון ש-"
שאל: האם יש "לכן" בסוף?
טענה vs.
משפט מפתח
טענה = עמדת הכותב (ניתן להסכים/לחלוק). משפט מפתח = הרעיון המרכזי של הפסקה, לא בהכרח עמדה.
טענה: "יש לאסור" / "הגורם הוא"
משפט מפתח: "פסקה זו עוסקת ב-"שאל: "האם הכותב לוקח עמדה?"
אם כן → טענה; אם רק מתאר → משפט מפתח

_💡 שאלת ההבחנה: "קרא את המשפט הראשון — האם הכותב כבר לוקח עמדה, או רק מציג מידע?"_

# **4. סוגי פתיחה**

לא כל פתיחה כוללת טענה מיד. יש לזהות את סוג הפתיחה לפני שמחפשים את הטענה.

**סוג הפתיחה**

**מאפיינים**

**סימני זיהוי**

**לשים לב**

פתיחה עם טענה
(Direct)
הטענה המרכזית של הכתיבה כולה מופיעה בפסקה הראשונה, לרוב במשפט 1-3. אקדמית.

"מחקר זה טוען" / "המאמר יראה" / "בניגוד לדעה הרווחת" — כבר בשורה הראשונה

הקורא יודע מיד לאן הכותב הולך. הפסקות הבאות = הצדקה לטענה.

פתיחה סיפורית
(Hook)
מתחילה בסיפור, נתון מפתיע, שאלה רטורית, תיאור סצנה, ציטוט. הטענה מגיעה מאוחר יותר.

שאלה רטורית / תיאור תמונה / נתון מפתיע / ציטוט — ואחרי כן: "המאמר יטען ש-"

הטענה עדיין קיימת — יש לחפש בסוף הפסקה הראשונה, לא בתחילתה.

פתיחת רקע
(Context)
מציגה הקשר, הגדרות, רקע לנושא. הטענה מגיעה אחרי שנוצר ההקשר. אקדמית מסורתית.

"בעשורים האחרונים" / "החוקרים הגדירו X" / "הדיון סביב" — לפני טענה

מבנה: רקע → פער / בעיה → טענה / שאלת מחקר.

_💡 טעות נפוצה: זיהוי המשפט הראשון כ"טענה" גם כשהוא רקע, נתון, או שאלה רטורית._

# **5. סוגי סגירה — סיכום, מסקנות, המלצות, סגירת מעגל**

ארבעה סוגי סגירה שונים בתפקידם. הכלי ידרוש מהסטודנט לזהות ולנמק.

**סוג הסגירה**

**הגדרה**

**מילות זיהוי**

**ההבדל מהאחרים**

סיכום
(Summary)
חזרה תמציתית על הנקודות שהוצגו. לא מוסיף מידע חדש.

"כפי שהוצג" / "כאמור לעיל" / "בפרק זה ראינו ש-"

מביט אחורה. לא לוקח עמדה חדשה. לא יוצר ידע חדש.

מסקנות
(Conclusions)
הסקת משמעות מהממצאים. עונה: מה ניתן ללמוד מכל שהוצג?

"מכאן ניתן להסיק" / "הממצאים מלמדים כי" / "לאור הנתונים"

מביט קדימה. מייצר ידע חדש — זו פרשנות, לא חזרה.

המלצות
(Recommendations)
הצעת פעולה / מדיניות / כיוון מחקר. עונה: מה לעשות עם המידע הזה?

"יש לשקול" / "מומלץ ל-" / "מחקר עתידי יוכל" / "המשמעות המעשית"

אקטיבי ומכוון פעולה. אופייני לכתיבה יישומית.

סגירת מעגל
(Circular Close)
חזרה לסיפור / שאלה / דימוי שבו נפתח הטקסט. יוצר שלמות.

חזרה לשם / לציטוט הפותח / לשאלה — עם תוספת משמעות חדשה

כוח רטורי ורגשי. אין חובה למסקנה מפורשת. עיתונאית.

_💡 שאלת ההבחנה: "האם הסגירה מוסיפה ידע חדש (מסקנה), קוראת לפעולה (המלצה), חוזרת על מה שנאמר (סיכום), או חוזרת לפתיחה (מעגל)?"_

# **6. סוגי חיזוק לטענה הראשית**

ביסוס טענה אינו רק "הוספת מידע" — יש סוגים בעלי חוזק שונה. הכלי ינחה לזהות ולהשתמש בנכון.

**סוג החיזוק**

**הגדרה**

**מילות זיהוי**

**חוזק ומגבלה**

**עובדה / נתון**

מידע מוסכם, מספרים, תוצאות מחקר הניתנים לאימות.

"מחקר מ-" / "X% מ-" / "מדידה הראתה" / "בסקר N נבדקים"

חזק — ניתן לאמת. תלוי באיכות המחקר ומהימנות המקור.

**ציטוט מומחה**

דברי חוקר / מדען / מומחה מוכר בתחום. מוסיף סמכות.

"לפי פרופ'" / "החוקר X מציין" / "כפי שכתב" / "על פי מחקרו"

משאיל סמכות. מומחה אחד אינו הוכחה — נדרש קונסנזוס.

**דוגמה ספציפית**

מקרה קונקרטי / אירוע / מצב שממחיש את הטענה.

"למשל" / "כפי שניכר במקרה" / "כך, ב-[שנה/מקום]"

ממחיש ומקרב. דוגמה אחת אינה מוכיחה כלל.

**טיעון לוגי**

הסבר הגיוני מדוע הטענה נכונה, ללא נתונים חיצוניים.

"שכן" / "זאת מפני ש-" / "מובן מאליו ש-" / "ניתן להסיק"

גמיש אך חלש לבדו. יעיל כשמשלים ראיה אמפירית.

**ניגוד / הפרכה**

הצגת הטענה הנגדית ואז הפרכתה. מחזקת הטענה ע"י דחיית ההתנגדות.

"יש הטוענים ש-... אולם" / "על אף ש-... בכל זאת"

מאוד חזק אקדמית — מראה שהכותב מכיר את שני הצדדים.

**אנלוגיה**

השוואה לתחום אחר להמחשת רעיון מופשט. פופולרי יותר מאקדמי.

"בדיוק כמו ש-" / "ניתן לדמות זאת ל-" / "בדומה ל-"

ממחיש ומושך. אנלוגיה אינה הוכחה — לא מספיקה לבדה.

_💡 כתיבה אקדמית חזקה משלבת לפחות שני סוגי חיזוק. עובדה + טיעון לוגי = יותר מכוח כל אחד לבדו._
