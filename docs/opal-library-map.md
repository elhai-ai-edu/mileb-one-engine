        שפה ראשונית ואוצר מילים
        Linguistic Foundation · מכינה שנה א׳ — מתחילים מוחלטים
      
    
    

      
        
          🔤
          המילה בהקשרWord-in-Context
        
        סטודנט מזין מילה → Opal מסביר פירוש, הטיות, שימוש במשפט, ותרגום לערבית/אנגלית/צרפתית
        
          מילה→
          פירוש + הטיות→
          משפט + תרגום
        
        מהגרים
      

      
        
          📊
          מסולם המיליםRegister Ladder
        
        סטודנט מגיש ביטוי יומיומי → Opal מחזיר 3 גרסאות: יומיומי / פורמלי / אקדמי עם הסבר
        
          מילה יומיומית→
          3 רמות שפה→
          הסבר הבדלים
        
        כלל
      

      
        
          📖
          מילון הקורסCourse Glossary
        
        סטודנט בוחר תחום (מדעי החברה / ביולוגיה) → Opal מייצר 20 מונחי מפתח עם הגדרה, משפט ותרגום
        
          תחום→
          20 מונחים→
          הגדרה + תרגום
        
        מהגרים
      

      
        
          🌉
          גשר הישיבהYeshiva-Bridge
        
        סטודנט מגיש ביטוי ישיבתי/יידיש → Opal מתרגם לאקדמית מודרנית ומסביר ההקשר החדש
        
          ביטוי ישיבתי→
          מקביל אקדמי→
          הסבר הקשר
        
        חרדים
      

      
        
          🔗
          ציד מילות הקישורConnector Hunt
        
        טקסט קצר → Opal מסמן מילות קישור לפי תפקיד: ניגוד / הוספה / סיבה / תוצאה / תנאי
        
          טקסט→
          סימון לפי צבע→
          הסבר תפקידים
        
        כלל
      

    
  

  

  
  
  
  
    
      2
      
        קריאה — מזיהוי פשוט לקריאה ביקורתית
        Reading Comprehension · משפט → פסקה → מאמר מלא
      
    
    

      
        
          🎯
          בלש הטענותClaim Detective
        
        סטודנט מזין פסקה → Opal שואל "מה הטענה?" → מקבל תשובה → מעריך: נכון / חלקי / שגוי + הסבר
        
          פסקה→
          שאלה סוקרטית→
          הערכה
        
        כלל
      

      
        
          🗂️
          פצחן המבנהStructure Decoder
        
        מאמר קצר → Opal ממפה: טענה ראשית / חיזוקים / מסקנה / מילות קישור בצבעים
        
          מאמר→
          מיפוי מבנה→
          תרשים ויזואלי
        
        כלל
      

      
        
          🔎
          קורא ביקורתיCritical Reader
        
        מאמר → Opal בודק: האם יש הנחות גלויות/נסתרות? האם הראיות מספיקות? מה חסר?
        
          מאמר→
          זיהוי הנחות→
          ניתוח חולשות
        
        מתקדמים
      

      
        
          🔍
          בודק פתיחה וסגירהFraming Checker
        
        מאמר → Opal מנתח רק הפסקה הראשונה והאחרונה: האם המעגל נסגר? ציון + הסבר
        
          פסקה ראשונה + אחרונה→
          ניתוח פתיחה/סגירה
        
        כלל
      

      
        
          🗣️
          קולות בטקסטVoice Identifier
        
        מאמר → Opal מזהה: מה דעת הכותב? מה דעת המקורות שהוא מצטט? האם יש מתח ביניהם?
        
          מאמר→
          מיפוי קולות→
          מתח/הסכמה
        
        מתקדמים
      

    
  

  

  
  
  
  
    
      3
      
        גשר — הכרת מבני טקסט וסוגות
        Text Types Bridge · בין קריאה לכתיבה
      
    
    

      
        
          🏗️
          מזהה הסוגהGenre Identifier
        
        סטודנט מגיש טקסט → Opal מזהה: טיעון / השוואה / תהליך / בעיה-פתרון / כתבה / פוסט + מסביר למה
        
          טקסט→
          זיהוי סוגה→
          סימני היכר
        
        כלל
      

      
        
          📐
          תבנית הסוגהGenre Template
        
        סטודנט בוחר סוגה (טיעון / ניגוד / תהליך...) → Opal מייצר תבנית עם הנחיות לכל חלק
        
          בחירת סוגה→
          תבנית מובנית→
          הנחיות לכל חלק
        
        כלל
      

      
        
          ⚖️
          השוואת סוגותGenre Comparison
        
        נושא → Opal כותב אותו ב-3 סוגות שונות (כתבה / מאמר / פוסט) + מסביר ההבדל בקהל ובמטרה
        
          נושא→
          3 גרסאות סוגה→
          ניתוח הבדלים
        
        כלל
      

    
  

  

  
  
  
  
    
      4
      
        כתיבה מובנית — ממשפט לחיבור
        Structured Writing · פסקה → חיבור → מאמר
      
    
    

      
        
          🎯
          מאמן משפט הפסקהTopic Sentence Coach
        
        סטודנט כותב משפט פסקה → Opal בודק: טענה ברורה? לא רחבה/צרה? + ניסוחים חלופיים
        
          משפט פסקה→
          אבחנה→
          חלופות
        
        כלל
      

      
        
          🔬
          שופט הלוגיקהLogic Judge
        
        פסקה → Opal מזהה: קפיצות לוגיות, סתירות פנימיות, משפטים לא מחוברים — דוח "אבחנה" מוסבר
        
          פסקה→
          ניתוח לוגיקה→
          דוח אבחנה
        
        כלל
      

      
        
          📝
          שדרוג מדרג שפהRegister Upgrader
        
        פסקה בשפה יומיומית → 3 גרסאות: בסיסי / אקדמי / מתקדם. מותאם לאוכלוסייה: גרסת מהגרים / חרדים
        
          שפה יומיומית→
          3 רמות→
          הסבר הבדלים
        
        מהגריםחרדים
      

      
        
          ✂️
          בודק פיסוקPunctuation Check
        
        פסקה → Opal מזהה שגיאות פיסוק, מסביר הכלל, ומציע גרסה מתוקנת עם הנמקה
        
          פסקה→
          זיהוי שגיאות→
          כלל + תיקון
        
        כלל
      

      
        
          🧱
          בונה החיבורEssay Builder
        
        סטודנט מזין נושא + עמדה → Opal מייצר שלד: פתיחה / 3 פסקאות + משפטי פסקה / סגירה
        
          נושא + עמדה→
          שלד חיבור→
          הנחיות לכל חלק
        
        כלל
      

      
        
          🔄
          ממיר הסגנון — חרדיStyle Converter
        
        טקסט בסגנון ישיבתי (פילפול, לשון ארוכה) → Opal ממיר לסגנון אקדמי תוך שמירת התוכן
        
          סגנון ישיבתי→
          סגנון אקדמי→
          הסבר שינויים
        
        חרדים
      

    
  

  

  
  
  
  
    
      5
      
        עבודה עם מקורות — מציטוט לאיגום
        Source Synthesis · מקור אחד → ריבוי מקורות → כתיבה משלבת
      
    
    

      
        
          📄
          מנתח המקורSource Analyzer
        
        קטע ממאמר → Opal: מה הטענה? מה הראיות? מי הכותב? מה הקהל המיועד? איך לצטט?
        
          קטע מאמר→
          ניתוח מלא→
          הנחיות ציטוט
        
        כלל
      

      
        
          🗃️
          מאגד המקורותSource Synthesizer
        
        3-5 קטעים + שאלת מחקר → Opal: תומכים / חולקים / משלימים → טבלת השוואה לסקירת ספרות
        
          קטעים + שאלה→
          מיון קולות→
          טבלת השוואה
        
        מתקדמים
      

      
        
          🖊️
          בודק הציטוטCitation Coach
        
        פסקה שנכתבה + מקורות → Opal בודק: האם המקורות שולבו נכון? ציטוט ישיר vs. עקיף? APA?
        
          פסקה + מקורות→
          בדיקת שילוב→
          תיקון + כלל
        
        כלל
      

      
        
          🧵
          הצמד הקולותVoice Weaver
        
        סטודנט כותב פסקה עם 2 מקורות → Opal בודק: האם המעבר בין הקולות חלק? האם יש "קולו" של הסטודנט?
        
          פסקה משלבת→
          בדיקת זרימה→
          חיזוק קול כותב
        
        מתקדמים
      

    
  

  

  
  
  
  
    
      6
      
        מחקר — כמותי, איכותני, סקירת ספרות
        Research Mastery · שאלת מחקר → מתודולוגיה → חיבור מלא
      
    
    

      
        
          🎯
          בונה שאלת מחקרResearch Q Builder
        
        תחום עניין רחב → דיאלוג סוקרטי של 5 שאלות → 3 גרסאות שאלת מחקר ממוקדת וניתנת לחקירה
        
          תחום עניין→
          5 שאלות צמצום→
          3 גרסאות שאלה
        
        מתקדמים
      

      
        
          🔬
          מזהה המתודולוגיהMethod Identifier
        
        שאלת מחקר → Opal מסביר: כמותי או איכותני? למה? מה הכלי המתאים (שאלון / ראיון / תצפית)?
        
          שאלת מחקר→
          כמותי/איכותני→
          המלצת כלי
        
        מתקדמים
      

      
        
          📚
          מנחה סקירת ספרותLit Review Guide
        
        מקורות + שאלת מחקר → Opal מייצר שלד לסקירת ספרות: נושאים / תיאוריות / פערים / מסקנה
        
          מקורות + שאלה→
          מיפוי נושאים→
          שלד סקירה
        
        מתקדמים
      

      
        
          📊
          בודק החיבור המחקריResearch Paper Reviewer
        
        טיוטת חיבור מחקרי → Opal: שאלת מחקר ברורה? מתודולוגיה עקבית? מסקנות נובעות? פערי ידע?
        
          טיוטה→
          4 קריטריונים→
          דוח פידבק
        
        מתקדמים
      

      
        
          🧩
          מנתח ראיון/תצפיתQualitative Analyzer
        
        תמלול ראיון / פרוטוקול תצפית → Opal מזהה קטגוריות, ציטוטים מייצגים, ודפוסים חוזרים
        
          תמלול/פרוטוקול→
          קטגוריות→
          דפוסים + ציטוטים
        
        מתקדמים
      

      
        
          📈
          פרשן הנתונים הכמותייםQuantitative Interpreter
        
        תוצאות סטטיסטיות (טבלה/ממצא) → Opal מסביר בשפה פשוטה ועוזר לנסח פסקת ממצאים
        
          נתונים/ממצאים→
          פרשנות→
          ניסוח אקדמי
        
        מתקדמים
      

      
        
          🏁
          מנחה החיבור המשלבSynthesis Mentor
        
        טיוטת חיבור משלב → בדיקה: מקורות מאוגמים ולא רק מצוטטים? עמדת הכותב ברורה? מסקנה מנומקת?
        
          חיבור משלב→
          3 קריטריונים→
          פידבק מובנה
        
        מתקדמים
