# מפת אוריינות אקדמית

  
ארכיטקטורת ידע מלאה · קריאה ← גשר → כתיבה · מחקר

   קריאה וניתוח
   גשר (תכנון וסוגות)
   כתיבה ועיצוב
   מחקר ואיגום
   אוצר מילים ושפה

  
  📖 קריאה וניתוח
  ✍️ כתיבה ועיצוב

  
  
    רמות הטקסט — קריאה
    מה אני קורא?
    
      
        1
        **משפט**זיהוי טענה, עובדה, דוגמה, מילת קישור
      
      
        2
        **פסקה**משפט פסקה + פיתוח + חיזוקים
      
      
        3
        **חיבור**פתיחה · גוף · סגירה · מבנה כולל
      
      
        4
        **טקסט שלם / מאמר**תקציר, מבוא, שיטה, ממצאים, דיון
      
    
  

  
    רמות הטקסט — כתיבה
    מה אני כותב?
    
      
        1
        **משפט**ניסוח מדויק, פיסוק, מדרג לשוני
      
      
        2
        **פסקה**בנייה נכונה של רעיון אחד שלם
      
      
        3
        **חיבור**תכנון, כתיבה, עריכה ושכתוב
      
      
        4
        **מאמר / סקירה**ניהול מקורות, ציטוט, סגנון אקדמי
      
    
  

  
  
    עומק הניתוח — קריאה
    כמה עמוק אני קורא?
    
      
        א
        **זיהוי פשוט**מה כתוב? מה הרעיון המרכזי?
      
      
        ב
        **זיהוי עמוק**למה המחבר בחר כך? מה מובלע?
      
      
        ג
        **קריאה ביקורתית**מה לא נאמר? מה הנחות היסוד? היכן הטיה?
      
    
  

  
    עומק הכתיבה
    כמה מורכבת הכתיבה שלי?
    
      
        א
        **תיאורי**מה? מי? מתי? — עובדות ותיאורים
      
      
        ב
        **מנתח**למה? כיצד? — קשרים וסיבות
      
      
        ג
        **ביקורתי-מסיק**לאן? מה המשמעות? — הסקה ועמדה
      
    
  

  
  
    
      גשר: תכנון הכתיבה
      לפני שכותבים — חושבים
      
        
          I
          **מיפוי רעיונות**סיעור מוחות, מפת חשיבה, רשימת טיעונים
        
        
          II
          **שלד כתיבה**כותרות + משפטי פסקה לפני המלל
        
        
          III
          **זיהוי קהל ומטרה**מי קורא? מה אני רוצה שיקרה?
        
      
    
    
      גשר: סוגות טקסט
      איזה סוג טקסט אני כותב?
      
        
          ✦
          **טיעון**עמדה + חיזוקים + התמודדות עם נגד
        
        
          ✦
          **השוואה / ניגוד**שתי אפשרויות זו מול זו
        
        
          ✦
          **תהליך / בעיה-פתרון**שלבים או אבחנה + המלצה
        
        
          ✦
          **כתבה / פוסט / מאמר**מטרות ז'אנריות שונות לכל סוג
        
      
    
  

  
  

    
      הכרת המילה
      מהי המילה?
      
        
          1
          **צורה ומשמעות**הגדרה, שורש, משפחת מילים
        
        
          2
          **הטיות**זמן, מין, מספר, גוף — שימוש נכון בפועל ובשם
        
        
          3
          **שימוש בהקשר**המילה בתוך משפט ובתוך פסקה — מתי נכון / לא נכון
        
      
    

    
      אוצר מילים אקדמי
      שפת הכתיבה האקדמית
      
        
          א
          **מילות קישור**לפי תפקיד: ניגוד, הוספה, סיבה, תוצאה, תנאי
        
        
          ב
          **פעלים אקדמיים**טוען, מראה, מנתח, מסיק, בוחן, מצביע על…
        
        
          ג
          **ניסוחים קונבנציונליים**פתיחות, סגירות, הסתייגויות — שגרות הטקסט האקדמי
        
      
    

    
      מדרג שפתי כללי
      שפה לפי רמה
      
        
          1
          **שפה יומיומית**מילים שכיחות, משפטים קצרים, ישירות
        
        
          2
          **שפה כתובה-פורמלית**מורכבות תחבירית, מילים פחות שכיחות
        
        
          3
          **שפה אקדמית-מתקדמת**מונחים דיסציפלינריים, מבנים פסיביים, דיוק מרבי
        
      
    

    
      לתלמידים עולים
      אוצר מילים לפי קורס
      
        
          ✦
          **מדעי החברה**חברה, מדיניות, כלכלה, ממשל, זהות
        
        
          ✦
          **מדעים מדויקים**נוסחה, תהליך, ניסוי, השערה, נתון
        
        
          ✦
          **הומניסטיקה**פרשנות, טקסט, משמעות, תרבות, זמן
        
      
    

  

  
  
    
      מקור יחיד
      עבודה עם מקור אחד
      
        
          1
          **סיכום נאמן**מה המקור אומר בלי דעתי
        
        
          2
          **ניתוח המקור**שיטה, טענות, חולשות
        
        
          3
          **הערכה ביקורתית**אמינות, רלוונטיות, מגבלות
        
      
    
    
      איגום מקורות
      חיבור בין מקורות
      
        
          1
          **השוואה**מה כל מקור אומר על X
        
        
          2
          **איגום**הסקת מסקנה ממספר מקורות
        
        
          3
          **כתיבה ממזגת**שלב + הסק + נמק ממקורות
        
      
    
    
      עולם המחקר
      מחקר עצמאי
      
        
          1
          **שאלת מחקר**מיקוד, ניסוח, ניתנת לחקירה
        
        
          2
          **כמותי / איכותני**הבדלים, שיטות, כלים
        
        
          3
          **סקירת ספרות**מחקר ← ניתוח ← סינתזה ← כתיבה
        
      
    
  

  
    
## 🔧 כלי Opal — מיפוי לפי תחום

    
כל כלי מקושר לרמה ספציפית במפה

  

  

    
      קריאה · רמת פסקה
      פצחן המבנה
      מדביקים פסקה, מקבלים: משפט פסקה + חיזוקים + מסקנה + מילות קישור מסומנות לפי תפקיד.
      קלט: פסקהפלט: תרשים
    

    
      קריאה · זיהוי עמוק
      בלש הטענות
      שיח סוקרטי — Opal שואל "מה הטענה?" מקבל תשובה ומשיב אם נכון/חלקי/שגוי עם הסבר.
      אינטראקטיביפידבק מיידי
    

    
      קריאה ביקורתית
      ציד ההנחות
      מנתח מה לא נאמר: הנחות יסוד, הטיות, פערים לוגיים — ומציג כשאלות לדיון כיתתי.
      קלט: מאמרפלט: שאלות
    

    
      כתיבה · רמת משפט
      מאמן המשפט
      מקבל משפט → בודק: האם יש טענה אחת ברורה? האם הניסוח אקדמי? מציע 3 ניסוחים חלופיים.
      קלט: משפטפלט: 3 גרסאות
    

    
      כתיבה · מדרג שפה
      מדרג השפה
      כותבים פסקה בשפה יומיומית — מקבלים 3 רמות: בסיסי / אקדמי / אקדמי-מתקדם + הסבר הבדלים.
      קלט: פסקהפלט: 3 רמות
    

    
      כתיבה · פיסוק ולוגיקה
      שופט הלוגיקה
      מזהה קפיצות לוגיות, משפטים לא מחוברים, סתירות פנימיות — מציג כ"דוח אבחנה" מוסבר.
      קלט: פסקה/חיבורפלט: דו"ח
    

    
      גשר · סוגות טקסט
      מזהה הסוגה
      מדביקים טקסט → Opal מזהה: איזה סוג טקסט? מה המבנה הצפוי? מה חסר לפי הסוגה הזו?
      קלט: טקסטפלט: ניתוח + המלצות
    

    
      גשר · תכנון
      בונה השלד
      מקבל נושא + סוגה → מייצר שלד כתיבה: כותרות + משפטי פסקה + מה צריך להיכנס לכל חלק.
      קלט: נושא + סוגהפלט: שלד
    

    
      מחקר · איגום מקורות
      מאגד המקורות
      3–5 קטעים + שאלת מחקר → טבלת השוואה: תומך / חולק / משלים — בסיס לסקירת ספרות.
      קלט: קטעים + שאלהפלט: טבלה
    

    
      מחקר · שאלת מחקר
      בונה השאלה
      דיאלוג סוקרטי של 5 שאלות — מצמצם עניין רחב לשאלת מחקר ממוקדת, ניתנת לחקירה ואקדמית.
      אינטראקטיביפלט: 3 גרסאות שאלה
    

    
      אוצר מילים · הקשר
      המילה בהקשר
      מקבל מילה → מציג 5 משפטים שמשתמשים בה נכון + 3 טעויות נפוצות בשימוש + הטיות עיקריות.
      קלט: מילהפלט: דוגמאות + שגיאות
    

    
      אוצר מילים · מדרג
      מסולם המילים
      מקבל רעיון פשוט → בונה סולם: מילה יומיומית / כתובה / אקדמית / מונח מקצועי — עם הסבר ההבדל.
      קלט: רעיוןפלט: סולם 4 רמות
    

    
      אוצר מילים · עולים
      מילון הקורס
      מקבל שם קורס + שפת אם → מייצר רשימת 20 מונחים מרכזיים עם הגדרה, דוגמה, ותרגום לשפת האם.
      קלט: קורס + שפהפלט: מילון מותאם
