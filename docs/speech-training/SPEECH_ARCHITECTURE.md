# Speech Training Architecture

## Overview
This document describes the layered architecture of the Speech Training Engine within MilEd.

## Layers
1. Speech Input Layer
2. Speech Metrics Engine
3. Role-Play Engine
4. Adaptive Engine
5. Profile Engine
6. Learning Journey Engine
7. Dashboard & Intervention

## Data Flow
Speech Input → Speech Data → Metrics → Roleplay State → Adaptive Decision → Profile Update → Journey Decision → Dashboard

## Design Principles
- Modular layers
- Non-breaking integration
- Config-first control
- Performance-based evaluation
