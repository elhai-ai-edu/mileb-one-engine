# Speech Training System

## Overview
The Speech Training Engine is a performance-based learning system designed to develop real-time communication skills through adaptive role-play simulations.

## Core Components
- Speech Input Layer
- Speech Metrics Engine
- Role-Play Engine
- Adaptive Engine
- Profile Engine
- Learning Journey Engine
- Dashboard & Intervention

## Key Principle
Speech is treated as performance data, not just text.

## Data Flow
Speech → Metrics → Roleplay → Adaptation → Profile → Journey → Dashboard
