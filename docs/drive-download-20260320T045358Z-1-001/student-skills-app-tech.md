# מדריך טכנולוגיות לפיתוח אפליקציית מיומנויות סטודנטים

## 1. שפות תכנות בסיסיות
### JavaScript
- שפה אוניברסלית לפיתוח אפליקציות
- קלה יחסית ללמידה למתחילים
- מאפשרת יצירת ממשקים דינמיים
- מתאימה לפיתוח WEB ומובייל

### Python 
- שפה ידידותית למתחילים
- חזקה בעיבוד נתונים ובינה מלאכותית
- קהילה גדולה ותמיכה נרחבת
- מאפשרת בניית אלגוריתמים מתקדמים

## 2. פריימוורקים לפיתוח אפליקציות
### React Native
- פיתוח אפליקציות למערכות iOS ו-Android
- קוד אחד עובד על שתי פלטפורמות
- ביצועים מהירים
- תמיכה קהילתית גדולה

### Flutter
- פיתוח מהיר של ממשקים
- עיצוב אחיד על פני פלטפורמות
- קלות בתחזוקת קוד
- יכולות עיצוב גבוהות

## 3. בינה מלאכותית והבנת למידה
### TensorFlow
- ספריית למידת מכונה של גוגל
- בניית מודלים לניבוי והתאמה אישית
- יכולות עיבוד נתונים מתקדמות
- תמיכה בלמידה עמוקה

### scikit-learn
- כלי לניתוח סטטיסטי
- זיהוי דפוסי למידה
- חיזוי הישגים אקדמיים
- קל יחסית לשימוש
