# שאלון מחקר: תפיסות סטודנטים ומרצים על מיומנויות להצלחה אקדמית

מחקר זה בוחן את תפיסות הסטודנטים והמרצים לגבי מיומנויות אקדמיות והצלחתם בלימודים.
השתתפותך בשאלון תסייע להבין אילו מיומנויות נתפסות כהכרחיות להצלחה בלימודים אקדמיים.
השאלון אנונימי ונתוניו ישמשו למטרות מחקר בלבד.
כדי להיכנס להגרלת הפרס, יש למלא את השאלון עד הסוף ולקבל מספר סידורי ייחודי.
הפרטים האישיים יישמרו חסויים לחלוטין ולא ישמשו לשום מטרה נוספת.
