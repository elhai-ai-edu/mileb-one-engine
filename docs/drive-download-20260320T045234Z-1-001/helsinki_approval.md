# בקשת אישור מוועדת הלסינקי

## 1. מטרת המחקר

מטרת המחקר היא לבדוק את השפעת סטרס אקדמי על סטודנטים, ואת האפשרות להשתמש בכלים דיגיטליים מבוססי AI להפחתת סטרס ושיפור רווחה נפשית.

## 2. שיטת המחקר

המחקר יכלול שני שלבים: שלב ראשון יבחן את רמות הסטרס, תחושת המסוגלות ותפיסות המיומנויות של הסטודנטים, ושלב שני יתמקד בהשפעת התערבות מבוססת טכנולוגיה על אותם מדדים.

## 3. פרטיות המשתתפים

כל הנתונים שייאספו יהיו אנונימיים, ולא יכללו מידע אישי מזהה. הנתונים יאוחסנו בצורה מאובטחת, ורק צוות המחקר יקבל גישה אליהם.

## 4. התמודדות עם משתתפים רגישים

המחקר כולל משתתפים שעשויים לחוות סטרס גבוה בשל חשיפה למלחמה. אנו נספק אפשרות לפרוש מהמחקר בכל שלב, ונציע תמיכה פסיכולוגית למשתתפים שיחושו מצוקה.

## 5. הסכמה מדעת

כל משתתף יקבל טופס הסכמה מדעת המפרט את מטרות המחקר, אופן איסוף הנתונים וזכותו לפרוש בכל עת.
