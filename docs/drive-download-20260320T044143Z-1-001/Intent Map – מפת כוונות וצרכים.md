השכבה השנייה מתרגמת את הפניות של הסטודנט לבוט – לשפה פדגוגית.
 היא מאפשרת לבוט להבין _מה עומד מאחורי המילים_,
 ולבחור תגובה מתאימה רגשית וקוגניטיבית, תוך קישור למשימות רלוונטיות מהשכבה הראשונה.

### **תוכן כל שורה**

כל אינטנט כולל:

- ביטויי פנייה נפוצים (איך הסטודנטים מתבטאים באופן טבעי)

- מילות מפתח לזיהוי אוטומטי

- צורך פדגוגי או רגשי (מה הבעיה האמיתית מאחורי המשפט)

- **4 תגובות חמות וזורמות** שהבוט יכול לבחור ביניהן

- קישורים ישירים למשימות המתאימות מה־Task Map

- **Priority_Level** (רגשי קודם לקוגניטיבי)

- Prompt מוכן לשילוב ב־GPT API (ליצירת תגובות בזמן אמת)
